# pptpd-centos7-install-script
Build your own VPN(pptpd) on centos7 server. I build it on my linode server which is centos7.


# How to use scripts

###1.Update centos7 yum
```
$ sh install-centos7-yum-pero.sh
```
###2.Install pptpd and run it
```
$ sh install-pptpd.sh
```
